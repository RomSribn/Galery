Мои работы

01

Пятнашки

Перемещая костяшки по коробке, добиться упорядочивания их по номерам, желательно сделав как можно меньше перемещений.

02

Косынка

В свободную ячейку (не дом) можно положить только короля. Игра заканчивается, когда все карты разложены. Цель игры состоит в том, чтобы разложить все карты в четыре стопки по возрастанию, начиная с туза так, чтобы карты одной масти находились в одной стопке.

03

Лабиринт

Сначала выберете желаемый размер поля ( Колонки | Строки ). Создайте поля, нажав "Create grid!" и с помощью клика выберете точку старта и финиша. После нажатия "Find path!", программа вычислит кратчайший путь ( если возможно )
